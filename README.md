## Run UI Preview (with stub backend)
1. Install dependencies.
  ```bash
  $ yarn
  ```

2. Configure.
  ```
  $ cp config/config.stub.json config/config.json
  ```

3. Run webpack-dev-server.
  ```bash
  $ yarn start
  ```

4. Open [http://localhost:8080/](http://localhost:8080/) with your browser.
